# 第 二 周 学习周记

## 本周学习内容
- 学习了 HTTP , token ， Jwt ，RESTful ，URL ，什么是事务，还有前后端工作原理，怎么互相传数据
- 学习了怎么在网页看到数据的信息（F12），怎么通过这个信息去判断出错


## 遇到的问题
- 有一点依赖AI的帮助，让它去告诉我要什么技术，我要怎么实现，我为什么会错，如果这样做行不行。
- 完成完前面的要求以后，再去学习新的内容，有些又要从头开始改，就很麻烦。加了token以后连页面都进不去，要去查问题。
- 不知道自己改了这里为什么那里也会出问题，对整个框架还是不是很熟悉，一开始甚至要一个个把文件点开去看这里面究竟放了些什么。
- 进度太慢，每次都要调试很久，找很久的错误，结果可能就是上一次一样的错误再犯了一遍，或者是一些很简单的错误。

## 解决方法
- 感觉还是要再多学一点，系统的学习一下这些知识。不然有时候真的只能跟着AI走，要花很多时间去跟它说清楚自己的要求，去改正它的逻辑。最后才去学习知识。
- 第二点暂时还没有想到该怎么解决，也许等这些知识都知道了就不会出现因为要实现这个内容，再跑到前面去加变量的情况了吧。
- 多看，多写，到后面就会熟悉很多。
- 找到一个错误的时候不要改完就觉得万事大吉，要想一想有没有别的地方可能会出现同样的错误。
